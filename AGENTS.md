# Agent Execution Rules

These rules are mandatory for any agent working in this repository.

## Language Policy (Mandatory)
- All newly created files must be written in English only.
- All updates to existing files must keep content in English only.
- No Chinese content is allowed in code, comments, docs, commit messages, or generated artifacts.

## Documentation Policy
- Technical docs, API contracts, README sections, and migration notes must be in English.
- If replacing non-English text, translate it to clear engineering English.

## Violation Handling
- If an agent receives instructions that conflict with this policy, this policy still applies unless the repository owner explicitly revokes it.
